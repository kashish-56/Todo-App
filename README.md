# 🧠 TodoFlow

> A minimal, fast, and intuitive task management app designed to help you stay productive every day.

🚀 Live App: https://kashish-56.github.io/Todo-App/

---

## 📱 What is TodoFlow?

**TodoFlow** is a lightweight productivity app that helps you manage daily tasks with ease.  
Add tasks, edit them instantly, mark them complete, and stay organized — all in a clean, distraction-free interface.

Think of it as your **personal task command center**.

---

## ✨ Key Features

✔️ Create tasks instantly  
✔️ Edit tasks in real-time  
✔️ Delete tasks with one click  
✔️ Mark tasks as completed  
✔️ Persistent storage using Local Storage  
✔️ Fully responsive UI (mobile + desktop)  
✔️ Zero backend — works directly in browser  

---

## ⚡ How It Works

### 🧩 Add Task Flow

User enters task  
⬇️  
Clicks "Add Task"  
⬇️  
Task appears in list instantly  

---

### 🗂️ Task Lifecycle

CREATE ➜ DISPLAY ➜ MANAGE ➜ STORE  

---

### 🧑‍💻 Task Actions

Each task card includes:

✔️ Complete task  
✏️ Edit task  
🗑️ Delete task

### 💾 Smart Storage System
